# Prerequisites

Before deploying Danny Tasks, ensure you have the following:

## Required Accounts

1. **GitHub Account**
   - Your repository should be hosted on GitHub
   - You'll need push access to deploy

2. **Platform Account** (choose one)
   - [Railway](https://railway.app) (recommended)
   - [Render](https://render.com)
   - [Fly.io](https://fly.io)

3. **API Keys**
   - [Todoist API Key](https://todoist.com/prefs/integrations)
   - [Claude API Key](https://console.anthropic.com/)

## Optional Accounts

- **Database Provider** (if using external Postgres)
  - [Neon](https://neon.tech)
  - [Supabase](https://supabase.com)

## System Requirements

### For Local Deployment
- Node.js 22+
- pnpm 8+
- Docker and Docker Compose (optional)

### For Platform Deployment
- No local requirements - everything runs in the cloud

## Next Steps

Once you have your accounts and API keys ready, proceed to the [deployment wizard](/deployment).
